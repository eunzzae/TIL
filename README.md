# TIL

이 프로젝트는 내가 공부한 것을 정리한 것입니다. 

[See Demo] (https://www.google.com/)

## Prerequisites

- python==3.11.0
- fastapi

## How to start
```shell
$ git clone {username/repo addr}
$ cd {repo addr}
$ python git/230915-git_시작하기.md
```

## Installation
```shell
$ pip install -r reqirements.txt
```

- 오늘 배운 내용
